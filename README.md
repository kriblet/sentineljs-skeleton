# sentineljs-skeleton
Example project for SentinelJS

### Get started
```
npm install -g sentineljs
git clone https://github.com/kriblet/sentineljs-skeleton.git
cd sentineljs-skeleton
sentinel start
```

**Navigate to http://localhost:8081/app**
